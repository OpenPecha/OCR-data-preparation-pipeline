# Image Orientation Labeler

Simple Mac app to label images as portrait or landscape.

## Usage

1. Install dependency:
```bash
pip install Pillow
```

2. Run:
```bash
python app.py
```

3. Click "Load Image Folder (b1)" and select your image folder
4. For each image, press **P** for Portrait or **L** for Landscape
5. Results saved to `orientations.json` in the same folder

## Keyboard Shortcuts

- `P` - Portrait
- `L` - Landscape
